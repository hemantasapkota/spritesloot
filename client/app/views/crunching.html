Open Source Box2D Editor

Download

Features

  • Sprietsheet Extraction
  • Entities Animation
  • Collision Shapes
  • Support for Physics Editor generated fixtures
  • Box2D Joints Support
  • Experimental Lua scripting support
  • Export to JSON

Technology

  • Eclipse RCP as a platform
  • Eclipse GEF for the Screen Editor
  • LibGdx for rendering in the desktop
  • Google Protobuf for file formats
  • LuaJ for executing lua scripts

Limitation

  • Cannot rotate entities in editor.
  • Joint support is basic.

About

We're building a marketplace for high quality animated sprites that integrates deeply with existing open-source game frameworks.
